VS code setup for playwright

1.	Open VS code
2.	Go to Extensions 
3.	Search Playwright (check that that publisher has verified ownership of microsoft.com)
4.	Press install
5.	Go to Explorer 
6.	In the command panel add: >install playwright 
7.	And choose which browsers you want to install ( you can choose all or only one)
